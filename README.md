# kviskr-downloads

Public release repository for Kviskr.

Kviskr is proprietary software. This repository does not contain the macOS app source code.

## What this repo is for

- GitHub Releases host signed `Kviskr.dmg` installers
- `appcast.xml` powers Sparkle auto-updates in the macOS app
- GitHub Issues can be used for bug reports and release-related feedback

## What this repo is not

- Not the private application source repository
- Not an open-source codebase
- Not the main place to learn how the app is implemented
